API Reference

Complete reference for all step types and features.

bash

Run shell commands.

{
  "type": "bash",
  "cmd": "npm run build",
  "timeout_ms": 60000,
  "retry": {
    "count": 3,
    "delay_ms": 1000,
    "backoff": true
  }
}

Fields

FieldTypeDescription
cmdstringShell command to execute (required)
idstringStep identifier
timeout_msnumberTimeout in milliseconds
retryobjectRetry configuration

read

Read file contents.

{
  "type": "read",
  "path": "src/main.rs",
  "max_bytes": 1048576,
  "encoding": "utf-8"
}

Fields

FieldTypeDescription
pathstringFile path to read (required)
max_bytesnumberMaximum bytes to read
encodingstringFile encoding (default: utf-8)

write

Write content to a file.

{
  "type": "write",
  "path": "output.txt",
  "content": "Hello World",
  "create_dirs": true
}

Fields

FieldTypeDescription
pathstringOutput file path (required)
contentstringContent to write (required)
create_dirsbooleanCreate parent directories (default: true)

grep

Search for patterns in files.

{
  "type": "grep",
  "pattern": "TODO",
  "path": "./src",
  "ext": ["rs", "js"],
  "regex": true,
  "context_lines": 2
}

Fields

FieldTypeDescription
patternstringSearch pattern (required)
pathstringDirectory to search (required)
extarrayFile extensions to filter
regexbooleanUse regex (default: false)
context_linesnumberLines before/after match

scan

Scan directory tree.

{
  "type": "scan",
  "path": "./src",
  "depth": 3,
  "include": ["*.rs", "*.toml"]
}

template

Render a template file.

{
  "type": "template",
  "builtin": "react-component",
  "output": "Component.tsx",
  "vars": {
    "name": "Button"
  }
}

if

Conditional step execution.

{
  "type": "if",
  "condition": { "type": "grep_has_results", "ref": 0 },
  "then": [
    { "type": "bash", "cmd": "echo found" }
  ],
  "else": [
    { "type": "bash", "cmd": "echo not found" }
  ]
}

each

Iterate over items.

{
  "type": "each",
  "over": ["a", "b", "c"],
  "as": "item",
  "parallel": false,
  "step": {
    "type": "bash",
    "cmd": "echo {{item}}"
  }
}

References

Pass data between steps using refs:

{
  "steps": [
    { "type": "scan", "path": "./src", "depth": 2 },
    { "type": "each", "over": {"ref": 0, "pick": "tree.*"}, "step": { "type": "bash", "cmd": "echo {{item}}" } }
  ]
}

Derived Transforms

Available in templates:

  • {{name | uppercase}}
  • {{name | lowercase}}
  • {{name | capitalize}}
  • {{name | pluralize}}
  • {{name | singularize}}
  • {{name | camelcase}}
  • {{name | snakecase}}
  • {{name | kebabcase}}
  • {{name | pascalcase}}
